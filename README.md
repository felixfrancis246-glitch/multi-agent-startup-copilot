# Multi-Agent Startup Copilot

A multi-agent AI system built using ArcByte Hosted n8n and Groq.

## Agents

1. CEO Agent
2. Research Agent
3. Sales Agent
4. Marketing Agent
5. Customer Support Agent
6. Project Manager Agent
7. Reporting Agent

## Workflow

User Input
→ CEO Agent
→ Research Agent
→ Sales Agent
→ Marketing Agent
→ Customer Support Agent
→ Project Manager Agent
→ Reporting Agent

## Files

- workflow.json.json
- AGENT_ARCHITECTURE.md
- PROMPTS.md
- TEST_RESULTS.md
- MULTI_AGENT_REPORT.md
